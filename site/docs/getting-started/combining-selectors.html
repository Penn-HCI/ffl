Selector Combinators

Like CSS, FFL offers ways to combine selectors for more precise selection. The following are the combinators supported by FFL. Note that the FFL combinators don't always have the exact same behaviors as their CSS counterparts. See below for more details.

Disjunction (OR)

You can join a list of Basic Selectors with commas (,) to match parts of the equation that match any of the selectors.

y=mx+b

Intersection (AND)

You can also write intersection(SELECTOR1,SELECTOR2,…) to match parts of the equation that match all of the selectors. This means overlaps of ranges selected by individual selectors (as opposed to the same range matching all selectors). The shorthand for this is to use zero or more whitespace (␣) to connect the selectors.

x^x_x

This is often used to disambiguate selections that might be too broad or difficult to express as a single basic selector.
